## Playground fall: lattice lane override only when column cell is **not** an edge neighbor

## Playground: T/V buttons — validate preset JSON before caching; retry fetch if cache bad

## Container editor: full crosshair (screen-aligned lines)

- **Files:** `Sandbox/editor.html`
- **What:** Replaced the mesh-cell crosshair tint with two faint SVG lines through the guide center. Endpoints are screen-aligned (horizontal/vertical in the viewport) after the editor rotation, then mapped into mesh space so the lines render correctly under the same `rotate` as the mesh.
