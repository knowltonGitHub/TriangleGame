Physics notes (expand)

Lane step: prefer empty downhill same-kind cells, min sideways offset to gravity line, then shortest along-gravity in tier. Skip rule: skip one mesh row in same column when not on container edge. Wall tip: if no lane step, prefer downhill rotation to boundary cell. Interior tip: otherwise downhill rotation with leftmost tie-break. Wall pivot: if already touching container wall, prefer downhill flip across a wall edge. Lay-flat: on boundary, prefer a downhill flip that increases full-edge wall contact before lane slide. Crash demo: ?crash=1 or crash-fall-demo.json.
